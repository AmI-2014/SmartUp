/ Vocal command functionality: performs the “speech to text” and handles interactions with microphone and speakers.
Modules exploiting this feature:

  • Recipe functionality: accessing the BigOven’s server through its API to fetch recipes to read
  • Music functionality: playing tracks or playlist filtered according to users’ request through the music web server Spotify
  • Alarm functionality: starting countdowns and alerting the user when time is up

/ Notifier functionality: users can ask to set the busy mode. When on the system collects all the notifications from social networks (Facebook, Twitter) or mailboxes (Gmail), reporting all the news when the status changes.
On user request it will also perform:
  • Birthday functionality: listing the user's Facebook friends with birthdays on the day
  • Agenda functionality: remebering to the user all commitments planned for the day (Google Calendar)
  • News functionality: responding with a summary of the day's top news headlines, sending them to the user over email if desire

The Smart Butler system will automatically select a subset of the listed functionalities to ease the performed user’s and, moreover, it will be able to set the environment with suitable temperature and/or music.
Users will be able to make the Smart Butler start tasks by using a list of key-words.
Some functionalities will be provided using the Jasper software.
